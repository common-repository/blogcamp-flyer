BlogCamp Flyer provides you a widget to promote future meetings of BlogCamp.

== Installation ==
The installation process is quite easy:

1. Simply upload the `blogcamp-flyer` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go in the `Widgets` menu to add the BlogCamp Flyer Widget
1. Configure your city and it's done :-)

Alternatively, you can also install the plugin through the Plugin Manager since WordPress 2.7.

== Frequently Asked Questions ==

= What is BlogCamp? =

BlogCamp is a meeting organization to promote good usages of blogging. The concept was launched in many spot in the world without links between them.
This plugin is provided by BlogCamp France and is willing to be spread all over the world.

= Why should I use this BlogCamp Flyer plugin? =

If you are a blogger (we guess so if you are on the WordPress plugins repository), you may want to promote a BlogCamp near you. It's a good way to meet new people, helping newbies to blog and share your knowledge.
It's the Web spirit ;-)
